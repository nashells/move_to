# ファイル移動プログラム

## 概要
Windows11のexplorer上でファイルを右クリックし、メニューから選択するとファイルを特定フォルダーに移動するプログラム。右クリックメニューとしては"move to"を表示し、さらにカスケードメニューとして目的のフォルダーを表すメニューが表示される。

## 要件詳細

### 対象ファイル
- ファイルの種類に制限なし
- 複数ファイルの同時選択・移動に対応

### 移動の動作
- 移動先に同盟ファイルが存在する場合：確認ダイアログを表示してユーザーに判断を委ねる
- 移動完了後のユーザーへの通知は不要
- 移動先フォルダが存在しない場合：自動作成せずにエラーダイアログを表示

### 移動先の設定
- メニューに表示される表示名と移動先フォルダーパスのペアリストが設定できる。
- 設定は設定ファイルに保存する
- 設定の編集は設定ファイルを直接編集する方式
- 登録できる移動先フォルダーの上限は10件

### 同名ファイル存在時の確認ダイアログ
- 選択可能なオプション：上書き、スキップ、リネーム、キャンセル
- 複数ファイル選択時の「すべてに適用」オプションは不要（ファイルごとに確認）

### エラーハンドリング
- 移動先フォルダーが存在しない場合：エラーダイアログを表示
- アクセス権限がない場合：エラーダイアログを表示

### 対象ファイル・フォルダー
- ファイルの種類に制限なし
- フォルダー（ディレクトリ）も移動対象に含まれる
- 複数ファイル・フォルダーの同時選択・移動に対応

## 開発テスト用設定

| メニュー表示名 | フォルダーパス |
| ------- | ------- |
| Temp    | C:\Temp |
