# UI設計ドキュメント

## 1. コンテキストメニュー（右クリックメニュー）

### 1.1 メインメニュー表示

エクスプローラーでファイルまたはフォルダーを右クリックした際に表示されるメニュー：

```
┌─────────────────────────┐
│ 開く                    │
│ 編集                    │
│ ───────────────────── │
│ move to              ▶ │  ← カスケードメニューあり
│ ───────────────────── │
│ コピー                  │
│ 切り取り                │
│ 削除                    │
│ 名前の変更              │
│ ...                     │
└─────────────────────────┘
```

### 1.2 カスケードメニュー（移動先選択）

「move to」にカーソルを合わせると表示されるサブメニュー：

```
┌─────────────────────────┐     ┌──────────────────┐
│ move to              ▶ │ ──▶ │ Temp             │
│                         │     │ Documents        │
│                         │     │ Archive          │
│                         │     │ Backup           │
│                         │     │ ...（最大10件）   │
└─────────────────────────┘     └──────────────────┘
```

- 設定ファイルに登録された表示名が一覧表示される
- 最大10件まで表示

## 2. 同名ファイル確認ダイアログ

移動先に同名ファイルが存在する場合に表示：

```
┌─────────────────────────────────────────────────────────┐
│  ⚠ ファイルの競合                                 [×]  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  移動先に同じ名前のファイルが既に存在します。          │
│                                                         │
│  ファイル名: example.txt                                │
│  移動先: C:\Temp                                        │
│                                                         │
│  どのように処理しますか？                               │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ ○ 上書き - 既存のファイルを置き換える          │   │
│  │ ○ スキップ - このファイルの移動をスキップする  │   │
│  │ ○ リネーム - 新しい名前を付けて移動する        │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│              [ OK ]           [ キャンセル ]            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### ボタン動作
| ボタン | 動作 |
|--------|------|
| OK | 選択したオプションで処理を続行 |
| キャンセル | 移動処理全体を中止 |

### オプション詳細
| オプション | 動作 |
|------------|------|
| 上書き | 移動先の既存ファイルを削除し、新しいファイルで置き換える |
| スキップ | このファイルの移動をスキップし、次のファイルの処理に進む |
| リネーム | リネームダイアログを表示し、新しい名前を入力させる |

## 3. リネームダイアログ

「リネーム」オプション選択時に表示：

```
┌─────────────────────────────────────────────────────────┐
│  📝 ファイル名の変更                              [×]  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  新しいファイル名を入力してください：                   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ example_copy.txt                                │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│              [ OK ]           [ キャンセル ]            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

- デフォルト値として「元のファイル名_copy.拡張子」を提案
- キャンセル時は同名ファイル確認ダイアログに戻る

## 5. 設定ファイル形式

ファイル名: `config.json`（想定）

```json
{
  "destinations": [
    {
      "displayName": "Temp",
      "path": "C:\\Temp"
    },
    {
      "displayName": "Documents",
      "path": "C:\\Users\\Username\\Documents"
    },
    {
      "displayName": "Archive",
      "path": "D:\\Archive"
    }
  ]
}
```

### 設定項目
| 項目 | 説明 |
|------|------|
| displayName | メニューに表示される名前 |
| path | 移動先フォルダーの絶対パス |

### 制約
- 最大10件まで登録可能
- 11件目以降は無視される
