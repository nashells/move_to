# ユーザーストーリー

## US-001: コンテキストメニューからファイルを移動する

**ユーザーとして**、エクスプローラーでファイルを右クリックして「move to」メニューから移動先を選択したい。  
**なぜなら**、よく使うフォルダーへ素早くファイルを移動できるようにするため。

### 受け入れ条件
- [ ] ファイルを右クリックすると「move to」メニューが表示される
- [ ] 「move to」にカーソルを合わせると、登録済みの移動先フォルダーがカスケードメニューとして表示される
- [ ] メニューから移動先を選択すると、ファイルがその場所に移動される
- [ ] 移動完了後、通知は表示されない

---

## US-002: コンテキストメニューからフォルダーを移動する

**ユーザーとして**、エクスプローラーでフォルダーを右クリックして「move to」メニューから移動先を選択したい。  
**なぜなら**、フォルダー単位でも素早く整理できるようにするため。

### 受け入れ条件
- [ ] フォルダーを右クリックすると「move to」メニューが表示される
- [ ] フォルダーがサブフォルダー・ファイルを含む場合も、まとめて移動される

---

## US-003: 複数ファイル・フォルダーを一括移動する

**ユーザーとして**、複数のファイルやフォルダーを選択して一括で移動したい。  
**なぜなら**、効率的にファイル整理を行いたいため。

### 受け入れ条件
- [ ] 複数のファイル・フォルダーを選択した状態で右クリックすると「move to」メニューが表示される
- [ ] 選択したすべてのアイテムが指定した移動先に移動される

---

## US-004: 同名ファイル存在時に対処方法を選択する

**ユーザーとして**、移動先に同名のファイルが存在する場合、どう対処するか選択したい。  
**なぜなら**、意図しない上書きを防ぎ、自分で判断したいため。

### 受け入れ条件
- [ ] 同名ファイルが存在する場合、確認ダイアログが表示される
- [ ] ダイアログには以下のオプションが表示される：
  - 上書き：既存ファイルを置き換える
  - スキップ：このファイルの移動をスキップする
  - リネーム：新しい名前を付けて移動する
  - キャンセル：移動処理を中止する
- [ ] 複数ファイル移動時は、同名ファイルごとに確認ダイアログが表示される

---

## US-005: 移動先フォルダーが存在しない場合のエラー表示

**ユーザーとして**、移動先フォルダーが存在しない場合、エラーメッセージを確認したい。  
**なぜなら**、問題を認識して設定を修正できるようにするため。

### 受け入れ条件
- [ ] 移動先フォルダーが存在しない場合、エラーダイアログが表示される
- [ ] エラーメッセージには存在しないフォルダーパスが含まれる

---

## US-006: アクセス権限エラーの表示

**ユーザーとして**、アクセス権限がない場合、エラーメッセージを確認したい。  
**なぜなら**、問題を認識して対処できるようにするため。

### 受け入れ条件
- [ ] 移動先へのアクセス権限がない場合、エラーダイアログが表示される
- [ ] エラーメッセージにはアクセス権限がない旨が含まれる

---

## US-007: 設定ファイルで移動先フォルダーを管理する

**ユーザーとして**、設定ファイルを編集して移動先フォルダーを追加・変更・削除したい。  
**なぜなら**、自分の用途に合わせてカスタマイズしたいため。

### 受け入れ条件
- [ ] 設定ファイルに表示名とフォルダーパスのペアを記述できる
- [ ] 設定ファイルを編集すると、メニューに反映される
- [ ] 最大10件まで登録できる
- [ ] 10件を超える設定は無視される（またはエラー表示）
